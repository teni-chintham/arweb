The Autoregressor

The Autoregressor is a tool that takes in a CSV file of your choice and uses statistical techniques and computer-based equations to help accomplish 3 main tasks!


The tasks:

Data Description will be given initally for a numerical understanding of data it is ok if you dont understand this there is always the use of a heatmap for better visualization.
Removal of data redundancy and data that may end up harming the overall larger data set.
This program will help to automatically conduct data analysis by creating various graphs based on the user's demands (Even if your knowledge of data is limited it is completely ok, cumulative graphs exist in this program to help give you an overall view of your data to help you as a user get a better idea of your data) .
Finally, the last and best function of this program is the ability to predict data for you this program will run your data through multiple regression models and output the best model with the best accuracy for you to predict data with if you wish to use a different type of regression with different settings a manual option to pick your regression model and setting is also present!!

Before Proceeding Here Are a Few Key Points You'll Need to Know:

The first part of this program is to understand its limitations. This program can only test against one value, the "Target." Don't worry if you already have a target value on your data set; the program will detect it and allow you to proceed with this target or change it. The second limitation is shown in the visual above. This program can only go as far as prediction but is unable to provide prescriptive information.
If you are unaware of the target, then simply put, it's the value you are testing for (Dependent Variable). For predictive analysis, this is vital as it needs a target value to predict. Example: In a dataset testing for CHD, the target value would be whether they have CHD or not, and the prediction will reveal the risk of heart disease.
The next vital focus is understanding regression. In simple terms, regression is the statistical process of finding a correlation between the target and independent variables through mathematical equations and statistical concepts.
There are a lot of regression models, so predictive analysis WILL TAKE TIME TO COMPUTE. This is due to the program finding the most accurate setting and type of regression for your data specifically.

This program uses 6 types of regression :

1.) Logistic Regression

Logistic regression is a method used to predict one of two possible outcomes, such as yes/no or pass/fail. It examines various factors to determine the likelihood of a specific outcome. For instance, it might look at factors like study hours and attendance to predict if a student will pass an exam. The model calculates a probability for each outcome. If the probability is over 50%, it predicts one outcome (e.g., the student will pass); if it's under 50%, it predicts the other outcome (e.g., the student will fail). It's widely used for its simplicity and effectiveness in making binary predictions.

2.) Lasso Regression

Lasso regression is a method used to improve the accuracy and interpretability of linear models. It does this by adding a penalty to the size of the coefficients of the model, effectively shrinking some coefficients to zero. This helps in selecting only the most important features.

Here's a simplified explanation:

Lasso regression predicts an outcome based on multiple input factors, but it also helps identify the most important factors by reducing the influence of less important ones. This makes the model simpler and helps prevent overfitting, where a model becomes too complex and performs poorly on new data. It’s useful when dealing with many factors and helps in building more reliable and interpretable models.

3.) Ridge Regression

Ridge regression is a technique used to improve the accuracy of linear models, especially when dealing with many correlated input factors. It adds a penalty to the size of the coefficients, but unlike Lasso regression, it doesn't shrink them to zero. Instead, it keeps all factors but reduces their impact to prevent overfitting.

Here's a simplified explanation:

Ridge regression predicts an outcome based on multiple factors, while also preventing the model from becoming too complex and overfitting the data. It does this by slightly reducing the influence of all factors, leading to a more reliable and stable model, especially when the input factors are highly correlated. It’s helpful in making more accurate predictions when dealing with many related factors.

4.) Random Forest Regression

Random forest regression is an ensemble learning method used for predicting continuous outcomes. It combines multiple decision trees to produce a more accurate and stable prediction. Each tree in the forest is built from a random subset of the data and features, and the final prediction is obtained by averaging the predictions of all the trees.

Here’s a simplified explanation:

Random forest regression predicts a continuous value (like house prices) by creating many decision trees using random subsets of data. Each tree gives its own prediction, and the final result is the average of all these predictions. This approach improves accuracy and reduces the risk of overfitting compared to using a single decision tree. It's powerful for handling large datasets and capturing complex relationships between variables.

5.) Support Vector Regression

Support Vector Regression (SVR) is a technique used for predicting continuous outcomes. It works by finding a hyperplane in a high-dimensional space that best fits the data points. SVR tries to minimize the error, which is the difference between the predicted and actual values, while also penalizing large deviations.

Here’s a simplified explanation:

Support Vector Regression predicts continuous values (like predicting house prices) by finding the best-fitting hyperplane in a high-dimensional space. It aims to minimize prediction errors and handles complex relationships between variables effectively. SVR is particularly useful when dealing with datasets with many features and nonlinear relationships.

6.) Multi-Layer Perceptron

MLP Regression, or Multi-Layer Perceptron Regression, is a type of neural network used for predicting continuous values. It consists of multiple layers of interconnected nodes (neurons), where each node applies a transformation to the input data. The network learns patterns in the data through training, adjusting the weights to minimize prediction errors.

Here’s a simplified explanation:

MLP Regression predicts continuous values (like house prices) using a neural network with multiple layers. Each layer processes the data, learning complex patterns and relationships. It adjusts the connections (weights) based on the data it sees, aiming to minimize the difference between its predictions and actual values. This makes it powerful for handling complex datasets with nonlinear relationships.
