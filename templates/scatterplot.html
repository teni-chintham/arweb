A scatter plot is a type of plot that shows the relationship between two variables.

What is a Scatter Plot?

A Scatter Plot uses dots to represent data points in a Cartesian plane.
It's used to assess relationships and trends between variables.

How to Read a Scatter Plot?

In a scatter plot, each dot represents a data point with its x-axis (horizontal) and y-axis (vertical) position.
Patterns in the scatter plot can indicate correlations or trends between variables.
