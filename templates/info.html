The Autoregressor

The Autoregressor is a tool that takes in a CSV file of your choice and uses statistical techniques and computer-based equations to help accomplish 3 main tasks!


The tasks:

Data Description will be given initally for a numerical understanding of data it is ok if you dont understand this there is always the use of a heatmap for better visualization.
Removal of data redundancy and data that may end up harming the overall larger data set.
This program will help to automatically conduct data analysis by creating various graphs based on the user's demands (Even if your knowledge of data is limited it is completely ok, cumulative graphs exist in this program to help give you an overall view of your data to help you as a user get a better idea of your data) .
Finally, the last and best function of this program is the ability to predict data for you this program will run your data through multiple regression models and output the best model with the best accuracy for you to predict data with if you wish to use a different type of regression with different settings a manual option to pick your regression model and setting is also present!!

This program uses 6 types of regression :

1.) Logistic
Regression

Logistic regression predicts binary outcomes (e.g., yes/no) by analyzing factors such as study hours to estimate the likelihood of a specific result. It calculates probabilities, predicting one outcome if the probability exceeds 50%, and the other if it doesnâ€™t. It's simple and effective for binary predictions.

2.) Lasso
Regression

Lasso regression improves linear models by adding a penalty to shrink some coefficients to zero, focusing on the most important features. This simplifies the model, preventing overfitting, and making it more reliable and interpretable, especially when dealing with many factors.

3.) Ridge
Regression

Ridge regression enhances linear models by slightly reducing all coefficient sizes to avoid overfitting, unlike Lasso regression, which can shrink some to zero. It maintains all factors, making the model more stable and accurate, especially with highly correlated inputs.

4.) Random
Forest Regression

Random forest regression predicts continuous outcomes by averaging results from multiple decision trees built from random data subsets. This method enhances accuracy and reduces overfitting compared to a single tree, effectively handling large datasets and complex relationships.

5.) Support Vector
Regression

Support Vector Regression predicts continuous values by finding the best-fitting hyperplane in high-dimensional space, minimizing prediction errors. It handles complex, nonlinear relationships effectively, making it suitable for datasets with many features.

Before Proceeding Here Are a Few Key Points You'll Need to Know:

The first part of this program is to understand its limitations. This program can only test against one value, the "Target." Don't worry if you already have a target value on your data set; the program will detect it and allow you to proceed with this target or change it. The second limitation is shown in the visual above. This program can only go as far as prediction but is unable to provide prescriptive information.
If you are unaware of the target, then simply put, it's the value you are testing for (Dependent Variable). For predictive analysis, this is vital as it needs a target value to predict. Example: In a dataset testing for CHD, the target value would be whether they have CHD or not, and the prediction will reveal the risk of heart disease.
The next vital focus is understanding regression. In simple terms, regression is the statistical process of finding a correlation between the target and independent variables through mathematical equations and statistical concepts.
There are a lot of regression models, so predictive analysis WILL TAKE TIME TO COMPUTE. This is due to the program finding the most accurate setting and type of regression for your data specifically.
