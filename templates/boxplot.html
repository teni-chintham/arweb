A box plot (or box-and-whisker plot) is a graphical representation of data distribution using quartiles.

What is a Box Plot?

A Box Plot summarizes the distribution of a continuous variable.
It shows the minimum, first quartile, median, third quartile, and maximum of the data.

How to Read a Box Plot?

In a box plot, the box represents the interquartile range (IQR) which contains the middle 50% of the data.
The whiskers extend from the box to show the range of the data excluding outliers.
